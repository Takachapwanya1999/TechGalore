# Technology Galore

This is a Vite + React project scaffolded for the 'Technology Galore' frontend.

## Getting Started

1. Install dependencies:
   ```powershell
   npm install
   ```
2. Start the development server:
   ```powershell
   npm run dev
   ```

## Project Structure
- `src/` - Main source code for the React app
- `public/` - Static assets
